Configura Firebase per salvare i dati nel cloud

  1. Vai su console.firebase.google.com → Crea progetto in modalità produzione
  2. Firestore Database → Crea database → Modalità produzione → regione eur3
  3. Vai su Authentication → Abilita Google come provider
  4. Vai su Firestore → Regole e incolla le regole qui sotto
  5. Vai su Impostazioni progetto → copia la config firebaseConfig → incollala nel file HTML
  6. Salva e ricarica — clicca "Accedi con Google" per entrare
Regole Firestore da incollare (Firestore → Regole → Pubblica):
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/data/{doc} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }
